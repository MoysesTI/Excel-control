Funções de Texto e Data

Domine o poder das funções de manipulação de texto e cálculos de data no Excel. Essas ferramentas poderosas são essenciais para análise de dados e automação de tarefas no dia a dia.

Funções de Texto

As funções de texto do Excel permitem manipular strings, extrair informações e formatar dados textuais para apresentação. Essenciais para limpar dados, gerar relatórios e personalizar a saída de informações.

CONCATENAR
=CONCATENAR(texto1; [texto2]; ...)
Une dois ou mais textos em um único texto. Ideal para combinar informações de diferentes células.
=CONCATENAR("Prisma";" ";"Arena")
→ "Prisma Arena"
MAIÚSCULA
=MAIÚSCULA(texto)
Converte todas as letras de um texto para maiúsculas, facilitando a padronização de dados.
=MAIÚSCULA("excel avançado")
→ "EXCEL AVANÇADO"
MINÚSCULA
=MINÚSCULA(texto)
Converte todas as letras de um texto para minúsculas, útil para comparação de textos.
=MINÚSCULA("PRISMA Arena")
→ "prisma arena"
PRI.MAIÚSCULA
=PRI.MAIÚSCULA(texto)
Converte a primeira letra de cada palavra para maiúscula e as demais para minúsculas.
=PRI.MAIÚSCULA("prisma ARENA")
→ "Prisma Arena"
TEXTO
=TEXTO(valor; formato)
Converte um número para texto com um formato específico. Perfeito para formatar valores monetários, datas e percentuais.
=TEXTO(1250,75;"R$ #.##0,00")
→ "R$ 1.250,75"
LOCALIZAR
=LOCALIZAR(texto_procurado; no_texto; [início])
Encontra a posição de um texto dentro de outro, considerando maiúsculas e minúsculas.
=LOCALIZAR("Excel";"Curso de Excel")
→ 10
PROCURAR
=PROCURAR(texto_procurado; no_texto; [início])
Similar ao LOCALIZAR, mas não diferencia maiúsculas de minúsculas na busca.
=PROCURAR("excel";"Curso de Excel")
→ 10
EXT.TEXTO
=EXT.TEXTO(texto; núm_inicial; [núm_caract])
Extrai caracteres de um texto a partir de uma posição específica.
=EXT.TEXTO("Prisma Arena";1;6)
→ "Prisma"
ESQUERDA
=ESQUERDA(texto; [núm_caract])
Extrai caracteres do início de um texto.
=ESQUERDA("Prisma Arena";6)
→ "Prisma"
DIREITA
=DIREITA(texto; [núm_caract])
Extrai caracteres do final de um texto.
=DIREITA("Prisma Arena";5)
→ "Arena"

Funções de Data

As funções de data permitem manipular, calcular e formatar informações temporais. Essenciais para análises de prazos, projetos e para automatizar relatórios com componentes temporais.

HOJE
=HOJE()
Retorna a data atual do sistema. Útil para cálculos automáticos baseados na data corrente.
=HOJE()
→ 22/03/2025 (depende da data atual)
AGORA
=AGORA()
Retorna a data e hora atuais do sistema, permitindo maior precisão em registros temporais.
=AGORA()
→ 22/03/2025 14:30 (exemplo)
DATA
=DATA(ano;mês;dia)
Cria uma data a partir de valores de ano, mês e dia, facilitando a criação de datas específicas.
=DATA(2025;3;21)
→ 21/03/2025
DIA
=DIA(data)
Extrai o dia do mês de uma data, retornando um valor entre 1 e 31.
=DIA("21/03/2025")
→ 21
MÊS
=MÊS(data)
Extrai o mês de uma data, retornando um valor entre 1 (janeiro) e 12 (dezembro).
=MÊS("21/03/2025")
→ 3
ANO
=ANO(data)
Extrai o ano de uma data, permitindo análises por período anual.
=ANO("21/03/2025")
→ 2025
DIAS
=DIAS(data_final;data_inicial)
Calcula o número de dias entre duas datas. Prático para calcular prazos e períodos.
=DIAS("21/03/2025";"01/01/2025")
→ 79
DATADIF
=DATADIF(data_inicial;data_final;"unidade")
Calcula a diferença entre duas datas em unidades específicas (dias, meses ou anos).
=DATADIF("01/01/2025";"21/03/2025";"M")
→ 2
DIA.DA.SEMANA
=DIA.DA.SEMANA(data;[tipo])
Retorna o dia da semana correspondente a uma data, como um número entre 1 (domingo) e 7 (sábado) por padrão.
=DIA.DA.SEMANA("21/03/2025")
→ 6 (sexta-feira)
DIATRABALHOTOTAL
=DIATRABALHOTOTAL(data_inicial;data_final;[feriados])
Calcula o número de dias úteis entre duas datas, excluindo finais de semana e feriados opcionais.
=DIATRABALHOTOTAL("01/01/2025";"21/03/2025")
→ 57 (aproximado)

Exercícios Práticos

Teste seus conhecimentos sobre funções de texto e data com estes exercícios interativos.

Exercício 1: Manipulação de Texto
Crie uma fórmula que extraia o código de produto a partir de um texto. Os códigos sempre começam com "PRD-" seguidos por 6 dígitos.
Dado o texto: "Item: PRD-123456 - Processador Quântico"
Você precisará usar LOCALIZAR para encontrar "PRD-" e EXT.TEXTO para extrair os 9 caracteres (incluindo "PRD-").
Exercício 2: Cálculo de Dias Úteis
Crie uma fórmula para calcular quantos dias úteis existem entre 01/03/2025 e 22/03/2025 (sem considerar feriados).
Calcule os dias úteis entre 01/03/2025 e 22/03/2025:
Use a função DIATRABALHOTOTAL que calcula automaticamente os dias úteis, excluindo finais de semana.
Exercício 3: Formatação de Dados Combinados
Crie uma fórmula que combine o nome de um produto, seu preço e data de validade em um formato específico.
Dados: "Visor Holográfico" (célula A1), 1299.90 (célula B1), 15/12/2025 (célula C1)
Resultado esperado: "VISOR HOLOGRÁFICO - R$ 1.299,90 (Validade: 15/12/2025)"
Combine MAIÚSCULA com CONCATENAR (ou &) e TEXTO para formatar o valor como moeda.
