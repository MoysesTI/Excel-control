Estrutura de Fórmulas
Todas as fórmulas no Excel começam com o sinal de igual (=) seguido por referências, operadores, valores e funções. O Excel segue a ordem padrão de operações matemáticas (PEMDAS): parênteses, exponenciação, multiplicação/divisão, adição/subtração.
=A1 + (B2 * C3) / D4
Operadores
Os operadores aritméticos incluem + (adição), - (subtração), * (multiplicação), / (divisão), ^ (exponenciação). Operadores de comparação incluem = (igual), <> (diferente), > (maior que), < (menor que), >= (maior ou igual), <= (menor ou igual).
=A1 > 100
Precedência de Operadores
Quando existem múltiplos operadores em uma fórmula, o Excel segue regras de precedência. Use parênteses para controlar a ordem de cálculo e garantir resultados corretos, especialmente em fórmulas complexas.
=(5+3)*2 resulta em 16
=5+3*2 resulta em 11
Funções Matemáticas
As funções matemáticas mais comuns incluem SOMA (para adicionar valores), MÉDIA (para calcular média aritmética), MÍNIMO (para encontrar o menor valor) e MÁXIMO (para encontrar o maior valor).
=SOMA(A1:A10)
=MÉDIA(B1:B20)
Funções de Texto
As funções de texto permitem manipular strings, como CONCATENAR (unir textos), ESQUERDA/DIREITA (extrair caracteres), MAIÚSCULA/MINÚSCULA (alterar capitalização) e TEXTO (formatar números como texto).
=CONCATENAR(A1, " ", B1)
=TEXTO(A2, "R$ #.##0,00")
Funções de Data
As funções de data e hora incluem HOJE (data atual), AGORA (data e hora atuais), DIAS (diferença entre datas), ANO/MÊS/DIA (extrair componentes de uma data) e FIMMÊS (último dia do mês).
=HOJE()
=DIAS(B2, A2)
Referências Relativas
Referências que se ajustam quando a fórmula é copiada para outras células. Por exemplo, se A1 contém =B1 e você copia para A2, a fórmula se ajusta para =B2. São úteis quando você quer aplicar o mesmo cálculo a diferentes linhas ou colunas.
A1, B2, C3
Referências Absolutas
Referências que não mudam quando a fórmula é copiada. Use o símbolo $ antes da letra da coluna e/ou número da linha. Pressione F4 para alternar entre tipos de referência. Útil para referências a células constantes como taxas ou valores fixos.
$A$1, $B$2, $C$3
Referências Mistas
Combinam elementos fixos e relativos. Em $A1, a coluna é absoluta mas a linha é relativa. Em A$1, a coluna é relativa mas a linha é absoluta. São úteis para cálculos matriciais e tabelas de consulta.
$A1, B$2, $C3
Tipos de Gráficos
O Excel oferece diversos tipos de gráficos, cada um adequado para diferentes dados e objetivos: Coluna/Barra (comparações), Linha (tendências temporais), Pizza (partes de um todo), Dispersão (correlações), Área (valores acumulados) e mais.
Inserir > Gráficos > [Tipo de Gráfico]
Elementos do Gráfico
Os principais elementos incluem Título, Eixos, Legendas, Rótulos de Dados, Linhas de Grade e Séries de Dados. Cada elemento pode ser personalizado individualmente para criar visualizações profissionais.
Clique no gráfico > Design > Adicionar Elemento de Gráfico
Personalização Visual
Explore as opções de Estilos Rápidos, Esquemas de Cores, e Layout para personalizar rapidamente seus gráficos. Para controle mais detalhado, use a guia Formatar para ajustar elementos individuais do gráfico.
Clique no gráfico > Design > Estilos de Gráfico
Estrutura de Tabela Dinâmica
Uma tabela dinâmica possui quatro áreas principais: Filtros (para filtrar o relatório inteiro), Colunas (para criar cabeçalhos horizontais), Linhas (para criar cabeçalhos verticais) e Valores (para dados agregados no corpo da tabela).
Inserir > Tabela Dinâmica
Cálculos e Resumos
Os valores podem ser resumidos usando diferentes funções: Soma, Contagem, Média, Máximo, Mínimo e mais. Também é possível criar campos calculados e mostrar valores como porcentagens ou diferenças.
Clique direito > Configurações do Campo de Valor
Ferramentas Interativas
Utilize Segmentações de Dados e Linhas do Tempo para criar controles interativos que filtram sua tabela dinâmica. Combine com gráficos dinâmicos para criar dashboards interativos e dinâmicos.
Analisar > Inserir Segmentação

Exercícios de Revisão

Pratique seus conhecimentos com estes exercícios que integram múltiplos conceitos abordados no curso.

Exercício 1: Relatório de Vendas
Crie um relatório de vendas completo que inclua fórmulas, funções, formatação condicional, gráficos e uma tabela dinâmica, aplicando os conceitos aprendidos nas aulas 1 a 10.
1
Crie uma tabela de dados de vendas com as colunas: Produto, Categoria, Região, Data, Quantidade, Preço Unitário, Total (fórmula) e Margem (fórmula).
2
Aplique formatação adequada: moeda para valores monetários, números para quantidades e formato personalizado para datas.
3
Crie fórmulas para calcular: Total (Quantidade * Preço), Margem (Total - Custo), Total Geral (SOMA) e Média de Vendas (MÉDIA).
4
Aplique formatação condicional para destacar produtos com alta margem (verde) e baixa margem (vermelho).
5
Crie um gráfico de colunas mostrando as vendas por região e um gráfico de linhas para tendências de vendas ao longo do tempo.
6
Crie uma tabela dinâmica que resuma as vendas por categoria e região, com valores mostrando soma e média das vendas.
Dica: Para criar referências dinâmicas no seu relatório, utilize a combinação de funções ÍNDICE e CORRESP para buscar valores em sua tabela de dados, permitindo atualizações automáticas quando novos dados forem adicionados.
Exercício 2: Dashboard Analítico
Construa um dashboard interativo que permita visualizar e analisar dados de diferentes perspectivas, integrando os conceitos de filtros, tabelas dinâmicas e gráficos.
1
Em uma nova planilha, crie uma área para controles de filtro usando segmentação de dados para filtrar por categoria, região e período.
2
Adicione uma tabela dinâmica que mostre vendas totais por produto e outra que mostre vendas por período.
3
Crie gráficos dinâmicos vinculados às tabelas dinâmicas: um gráfico de barras para produtos e um gráfico de linha para tendência temporal.
4
Adicione indicadores de desempenho que mostrem total de vendas, média por transação e contagem de vendas, usando funções como SOMA, MÉDIA e CONT.SE.
5
Utilize formatação condicional para destacar visualmente os indicadores (verde para acima da meta, vermelho para abaixo).
6
Adicione uma fórmula de cálculo de data usando HOJE() que mostre há quantos dias ocorreu a última venda.
Dica: Conecte todas as segmentações a todas as tabelas dinâmicas para garantir que os filtros afetem tanto as tabelas quanto os gráficos simultaneamente, criando uma experiência verdadeiramente interativa.
